---

Analyze the project in the current directory and run `/init` to create or improve the AGENTS.md file.

Then run a deep analysis of:
1. Project structure and architecture
2. Build system and dependencies
3. Test setup and coverage
4. Linting and formatting configuration
5. CI/CD setup if present

Suggest improvements and implement any quick wins (missing config files, outdated configs, etc.).
