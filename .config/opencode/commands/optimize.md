Analyze the current codebase for performance optimization opportunities:
1. Identify expensive operations (O(n²) or worse)
2. Look for unnecessary allocations/copies
3. Check for redundant API calls or database queries
4. Identify code that could benefit from caching or memoization
5. Find bundle size issues (large dependencies, missing tree-shaking)

For each finding, explain the issue, suggest a fix, and quantify the expected improvement.
